About This Project

This repository demonstrates how to use Constitution Files (steering documents) to create consistent, high-quality prototypes and production code with AI coding assistants.

Below are live, interactive prototypes created using our prototype constitution standards. Each prototype is a single HTML file with no build tools required.

Complete CRM dashboard with metrics, revenue charts, customer list, contact management, and export functionality.

Features:

  • Key metrics cards
  • Revenue trend chart

  • Contact management
  • Export to CSV/Excel

How These Were Created

These prototypes were generated using AI with our prototype-standards.md constitution file, which defines:

  • • HTML5 + Tailwind CSS + Vanilla JavaScript
  • • Single file, no build tools required
  • • Professional, modern design system
  • • Mock data and business logic patterns
  • • Ready for stakeholder demos
